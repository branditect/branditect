# Branditect

**AI-powered brand operating system.**

Branditect gives every company a unified workspace — *Branditect for [Brand Name]* — where teams can manage brand identity, assets, guidelines, and creative output with the help of AI.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Auth & Database:** Supabase
- **Hosting:** Vercel (planned)

## Getting Started

```bash
# Install dependencies
npm install

# Copy env template and fill in your Supabase credentials
cp .env.local.example .env.local

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

```
app/            → Next.js App Router pages and layouts
components/     → Reusable React components
lib/            → Utilities and service clients (Supabase, etc.)
styles/         → Additional stylesheets
```
